Class: Endboss

Endboss()

The Endboss class represents the main boss enemy in the game. It extends the MovableObject class and defines specific behaviors such as animations for walking, getting hurt, and dying.

Constructs an Endboss object by loading the walking, hurt, and dead images. Sets the starting position of the Endboss and initiates movement.
