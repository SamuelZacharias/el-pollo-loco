Class: Coin

Coin()

Represents a coin object in the game, which can be collected by the player. Extends the Drawableobject class to handle image loading and rendering functionality.

Constructor

new Coin()

Constructs a new coin object, setting its position with a minimum distance from other coins and loading the corresponding image.

setPositionWithMinDistance(minX, maxX, minDistance)

Sets the position of the coin along the X axis, ensuring a minimum distance from other coins. The method tries a number of random positions until the coin is far enough from others.
