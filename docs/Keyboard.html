Class: Keyboard

Keyboard()

The Keyboard class represents the state of various key presses used to control the character in the game. It tracks whether keys like LEFT, RIGHT, UP, DOWN, SPACE, and Q are being pressed or not. This class does not have any methods, as it primarily serves as a data structure to hold the boolean states of each key.
